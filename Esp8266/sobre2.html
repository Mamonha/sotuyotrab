Sobre

O sistema foi desenvolvido por uma equipe de Alunos do curso de Engenharia Mecânica da Universidade Estadual do Paraná Campus Foz do Iguaçu, orientado pelo Doutor Juan Carlos Sotuyo;

Funcionamento

Atua como um sistema de leitura biométrica em transportes públicos, com intuito de diminuir problemas que ocorrem com o sistema atual de cartões da região da cidade de Foz do Iguaçu;
O funcionamento técnico do sistema está apresentado abaixo:

Imagem 1- Placa Wemos D1 R2

Placa Wemos D1 R2:

O sistema é controlado por uma placa microcontroladora, operando com um módulo wi-fi ESP-8266, esse dispositivo é responsável por realizar uma conexão wi-fi, e posteriormente uma conexão em uma base de dados em que as informações do usuário serão armazenadas.
Além disso essa placa é responsável por comandar os outros dispositivos acoplados a ela, descritos a seguir;

Imagem 2- Módulo de Biometria DY50;

Leitor Biométrico Dy50:

Para a coleta das biometrias foi utilizado um módulo
de impressão digital DY50. No sistema ele coleta as biometrias
para realizar um cadastro de um usuário novo, e posteriormente
para acesso ao sistema;

Imagem 3- Display LCD 16X2

Display LCD:

Um tipo de visor utilizado para mostrar mensagens no intuito de auxiliar o usuário a utilizar o sistema, é apresentado algumas
mensagens na utilização da coleta de biometria;
